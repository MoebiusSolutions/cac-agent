Using CAC Agent with Older JGit Releases

Using cac-agent with Older JGit Releases

The startup code for jgit.sh is a good example of using this agent:

#!/bin/sh
# Git bash

dir="$(cd "$(dirname "$0")" && pwd -W)"

java \
-Xmx512m \
-javaagent:"$dir"/cac-agent.jar=load \
-jar "$dir"/jgit-cli.jar $@

With the above script in place, you can clone a remote repository that requires a CAC with

jgit.sh clone <url>

When you are ready to push you can use

jgit.sh push [repository]

Fetching updates

jgit.sh fetch [repository]
